Mission Statement:

We love to code. We love building and creating fun games and programs in Javascript, CSS and HTML that can work on any mobile or desktop browser. Our mission is to have fun while teaching and learning coding.

About:

Comical Coders is a coding club at Frank S. Greene Middle School, run by students for students. The club is lead by Crystal, David, Julian, and Vince. Each club member is given their own personal portfolio where he/she work on personal coding projects. Comical Coders is mentored by Tasha and sponsored by Wisen Space.
